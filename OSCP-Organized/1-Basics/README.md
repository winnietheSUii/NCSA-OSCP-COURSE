# 1-Basics

Foundation concepts and essential knowledge for penetration testing.

## Topics Covered

- **DNS Fundamentals**: Understanding DNS queries, records, and enumeration
- **Directory Listing**: Web server directory enumeration techniques
- **Robots.txt**: Search engine exclusion files and their security implications
- **JavaScript Login Bypass**: Client-side authentication bypass methods
- **Autocomplete**: Form autocomplete vulnerabilities and exploitation
- **Backup Lab**: Lab environment setup and backup considerations

## Quick Commands

```bash
# List all topics
ls -la

# Find specific content
grep -r "DNS\|directory\|robots" . --include="*.md"

# View a specific file
cat "filename.md"
```

## Learning Path

1. Start with DNS fundamentals
2. Understand directory enumeration
3. Learn about robots.txt implications
4. Study JavaScript-based authentication
5. Practice with autocomplete vulnerabilities
